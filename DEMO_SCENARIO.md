# Orbit AI 시연 시나리오

## 🎯 시연 목표
Orbit AI가 유학생들을 위한 AI 기반 강의 번역 및 학습 플랫폼임을 보여주고, 핵심 기능들을 자연스럽게 시연합니다.

---

## ⏱️ 시연 시간: 약 5-7분

---

## 📋 시연 시나리오

### 1️⃣ **오프닝 (30초)**

**대사:**
> "안녕하세요. Orbit AI를 소개합니다. Orbit AI는 유학생들을 위한 AI 기반 강의 번역 및 학습 플랫폼입니다. 이해하기 어려운 강의를 업로드하면, 자동으로 번역, 요약, 퀴즈, 리뷰를 생성해주는 서비스입니다."

**액션:**
- 로그인 페이지 화면 보여주기
- 언어 선택 버튼 클릭 (한국어 → 영어 → 중국어 등) → 다국어 지원 강조

---

### 2️⃣ **회원가입 및 로그인 (1분)**

**대사:**
> "먼저 회원가입을 진행하겠습니다. Orbit AI는 5개 언어를 지원합니다: 한국어, 영어, 중국어, 일본어, 베트남어. 사용자의 모국어를 선택하면 모든 인터페이스가 해당 언어로 표시됩니다."

**액션:**
1. "회원가입" 버튼 클릭
2. 회원가입 폼 작성:
   - 이름: "홍길동"
   - 사용자 ID: "hong123"
   - 이메일: "hong@example.com"
   - 비밀번호: "password123"
   - 언어: "한국어" 선택
3. "회원가입" 버튼 클릭
4. 성공 토스트 메시지 확인 (언어별로 표시됨)
5. 로그인 페이지로 이동
6. 로그인 정보 입력 후 로그인

**강조 포인트:**
- ✅ 다국어 지원 (5개 언어)
- ✅ 직관적인 UI/UX
- ✅ 언어별 맞춤 인터페이스

---

### 3️⃣ **폴더 생성 및 강의 업로드 (1.5분)**

**대사:**
> "로그인 후 홈 화면으로 이동했습니다. 먼저 강의를 정리할 폴더를 생성하겠습니다."

**액션:**
1. 사이드바에서 "새 폴더 만들기" 클릭
2. 폴더 이름 입력: "컴퓨터과학 기초"
3. 폴더 생성 확인
4. 생성된 폴더 선택

**대사:**
> "이제 강의 파일을 업로드하겠습니다. 음성 또는 영상 파일을 업로드하면 AI가 자동으로 처리합니다."

**액션:**
1. 업로드 영역 클릭 또는 파일 드래그 앤 드롭
2. 강의 파일 선택 (예: "lecture_sample.mp3")
3. 업로드 모달에서:
   - 폴더 선택: "컴퓨터과학 기초"
   - 수업 제목 입력: "데이터 구조와 알고리즘 1강"
4. "확인" 버튼 클릭
5. 업로드 진행 상황 표시 (스켈레톤 UI)
   - "파일을 업로드하고 있습니다..."
   - "파일을 처리하고 있습니다..."
   - "번역을 진행하고 있습니다..."
   - "완료 중입니다..."
6. 완료 메시지: "수업 내용의 번역이 완료되었어요!"

**강조 포인트:**
- ✅ 직관적인 파일 업로드
- ✅ 실시간 진행 상황 표시
- ✅ 자동 번역 처리

---

### 4️⃣ **번역본 및 요약 확인 (1분)**

**대사:**
> "강의가 업로드되고 처리되었습니다. 이제 강의 상세 페이지로 이동했습니다. 왼쪽에는 번역본이, 오른쪽에는 요약, 퀴즈, 리뷰 탭이 있습니다."

**액션:**
1. 강의 상세 페이지 자동 이동 확인
2. 왼쪽: 번역본 내용 스크롤하며 보여주기
3. 오른쪽: "요약" 탭 클릭
4. 요약 내용 확인 (마크다운 형식)
5. 요약 내용 스크롤하며 보여주기

**강조 포인트:**
- ✅ 양방향 레이아웃 (번역본 + 학습 콘텐츠)
- ✅ 자동 생성된 요약
- ✅ 깔끔한 UI

---

### 5️⃣ **퀴즈 생성 및 확인 (2분)**

**대사:**
> "이제 퀴즈를 생성해보겠습니다. 퀴즈는 단답형, O/X, 객관식 세 가지 유형을 지원하며, 난이도도 선택할 수 있습니다."

**액션:**
1. "퀴즈" 탭 클릭
2. "퀴즈 생성" 버튼 클릭
3. 퀴즈 생성 폼 작성:
   - 문제 개수: 5개
   - 난이도: "중" 선택
   - 문제 유형: "객관식" 선택
4. "문제지 생성" 버튼 클릭
5. 퀴즈 생성 진행 상황 표시:
   - "강의 내용을 분석하고 있습니다..."
   - "문제를 생성하고 있습니다..."
   - "난이도를 조정하고 있습니다..."
   - "최종 검토 중입니다..."
6. 생성 완료 후 퀴즈 리스트 표시
7. 퀴즈 항목 클릭하여 상세 페이지 이동

**대사:**
> "퀴즈 상세 페이지로 이동했습니다. 왼쪽에는 번역본, 오른쪽에는 생성된 퀴즈가 표시됩니다."

**액션:**
1. 퀴즈 상세 페이지 확인
2. 문제 확인 (객관식 문제 예시)
3. "답 보기" 버튼 클릭하여 정답 확인
4. 여러 문제 스크롤하며 보여주기

**강조 포인트:**
- ✅ 다양한 문제 유형 지원
- ✅ 난이도 조절 가능
- ✅ 실시간 퀴즈 생성
- ✅ 직관적인 답 확인 기능

---

### 6️⃣ **리뷰 생성 (1분)**

**대사:**
> "마지막으로 리뷰 기능을 보여드리겠습니다. 리뷰는 강의 내용을 더 깊이 이해할 수 있도록 도와주는 심화 학습 콘텐츠입니다."

**액션:**
1. 강의 상세 페이지로 돌아가기
2. "리뷰" 탭 클릭
3. "리뷰 생성" 버튼 클릭
4. 리뷰 생성 진행 상황 표시
5. 생성된 리뷰 내용 확인 (마크다운 형식)
6. 리뷰 내용 스크롤하며 보여주기

**강조 포인트:**
- ✅ 심화 학습 콘텐츠 제공
- ✅ 자동 생성된 리뷰

---

### 7️⃣ **Tips 페이지 및 언어 변경 (30초)**

**대사:**
> "Orbit AI는 학교 생활에 유용한 Tips도 제공합니다."

**액션:**
1. 사이드바에서 "학교 꿀팁" 클릭
2. Tips 페이지 확인
3. 카테고리별 Tips 내용 확인
4. 언어 선택 버튼 클릭 (한국어 → 영어)
5. 인터페이스가 영어로 변경되는 것 확인

**강조 포인트:**
- ✅ 실용적인 Tips 제공
- ✅ 실시간 언어 변경

---

### 8️⃣ **마무리 (30초)**

**대사:**
> "이상으로 Orbit AI의 주요 기능을 시연했습니다. Orbit AI는 유학생들이 어려운 강의를 이해하고 학습할 수 있도록 돕는 AI 기반 플랫폼입니다. 감사합니다."

**액션:**
- 전체 화면 한 번 더 보여주기
- Q&A 준비

## 🎬 시연 팁

### ✅ 준비 사항
1. **테스트 계정 준비**
   - 회원가입용 계정 정보 미리 준비
   - 로그인 정보 확인

2. **테스트 파일 준비**
   - 강의 샘플 파일 (음성/영상) 준비
   - 파일 크기 확인 (100MB 이하)

3. **인터넷 연결 확인**
   - API 서버 연결 상태 확인
   - 네트워크 안정성 확인

4. **브라우저 준비**
   - Chrome 또는 Edge 권장
   - 개발자 도구 열어두기 (에러 확인용)

### ⚠️ 주의사항
- API 응답 시간이 길 수 있으니 여유있게 진행
- 에러 발생 시 자연스럽게 넘어가기
- 핵심 기능에 집중하기

### 💡 강조할 포인트
1. **다국어 지원** - 5개 언어 완벽 지원
2. **자동화** - 업로드만 하면 자동 처리
3. **학습 지원** - 번역, 요약, 퀴즈, 리뷰 통합 제공
4. **사용자 친화적 UI** - 직관적이고 깔끔한 인터페이스
